A compact setting that allows choosing a color. It consists of a color preview that shows the current color and a Choose button. Clicking on the button opens a new menu with a color picker.

This type inherits the following: [mwseMCMColorPicker](../types/mwseMCMColorPicker.md), [mwseMCMSetting](../types/mwseMCMSetting.md), [mwseMCMComponent](../types/mwseMCMComponent.md)

!!! tip
