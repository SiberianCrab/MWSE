# Dialogue Pages

!!! tip
	These values are available in Lua by their index in the `tes3.dialoguePage.voice` table. For example, `tes3.dialoguePage.voice.hello` has a value of `0`.

Index    | Value
-------- | -----
hello    | `0`
idle     | `1`
intruder | `2`
thief    | `3`
hit      | `4`
attack   | `5`
flee     | `6`
